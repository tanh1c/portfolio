## 🔮 Tech Stack
- **Framework Core**: React 19 + TypeScript + Vite
- **Styling Matrix**: Tailwind CSS v4
- **Animation Engine**: Framer Motion
- **Identities**: Apple Emojis, iOS style interface cues

## 📡 Initialization (Local Development)

To boot up the portfolio directly on your local station:

```bash
# 1. Clone the repository
git clone https://github.com/tanh1c/portfolio.git
cd portfolio

# 2. Install necessary dependencies
npm install

# 3. Ignite the development server
npm run dev
```

Your system will lock onto local port `http://localhost:3000` (or `http://localhost:5173`).
